# Darna Android App (Frontend)

This is the **Android frontend** for the Darna application. The project is built with **Kotlin** and **Jetpack Compose**, targeting Android devices. It allows users to browse houses, view details, and add new announcements.

## Features

- Browse a list of houses with details (price, location, tags, calm level, lifestyle, energy, availability)
- 360° house visit UI badge
- Add new house announcements
- View house details
- Jetpack Compose UI components for a modern Android UI
- Integration with backend API via Retrofit

## Screens

- **Home Screen** – Displays the list of houses with basic info and tags.
- **Details Screen** – Shows detailed information about a selected house.
- **Add Annonce Screen** – Allows adding a new house announcement.
